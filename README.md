# 只放羊（Vanilla HTML/CSS/JS）

## 預覽（最短步驟）
1. 下載此倉庫。
2. 用 Chrome 直接開啟 `index.html`。

## 素材放置
- 背景影片：`assets/bg/grassland.MP4`
- 羊素材 6 幀：
  - `assets/sheep/sheep_rf1.png`
  - `assets/sheep/sheep_rf2.png`
  - `assets/sheep/sheep_rf3.png`
  - `assets/sheep/sheep_rf4.png`
  - `assets/sheep/sheep_rf5.png`
  - `assets/sheep/sheep_rf6.png`

> 素材放上去後會自動啟用；若尚未放素材，仍可直接執行（背景與羊都會使用 fallback）。
